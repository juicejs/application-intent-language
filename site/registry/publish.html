Publish A Feature Package

  1. Create registry/packages/<feature>/.
  2. Add package.json and root-level .ail files.
  3. Use valid AIL preambles in all files.
  4. Open a pull request.

Minimal manifest:

{
  "name": "juice.games.snake",
  "version": "1.3",
  "summary": "Single-player snake with leaderboard",
  "files": [
    "juice.games.snake.intent.ail"
  ]
}

CI validates package structure and AIL header identity on every PR. See registry/README.md.
