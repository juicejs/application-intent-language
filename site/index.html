Application Intent Language (AIL)

Intent-first feature specification for humans and AI coding agents. Start with one intent file, add precision facets only when needed.

Read Specification Browse Registry Repository README

Canonical Shape

Each feature uses a package-facet preamble:

AIL: <feature>#<facet>@<x.y>

Minimal feature entrypoint is <feature>.intent.ail.

Community Registry

Feature packages live under /registry/packages and are published through pull requests with automated validation.
